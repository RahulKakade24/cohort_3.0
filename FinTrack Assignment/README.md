# 💰 FinTrack Pro

FinTrack Pro is a modern personal finance tracker built using **HTML5, CSS3, and Vanilla JavaScript**. It helps users manage their daily income and expenses through a clean, responsive, and interactive interface. All data is stored locally in the browser using the **Local Storage API**, so no backend or database is required.

## ✨ Features

- User Registration & Login
- Multiple User Accounts
- Add Income & Expense Transactions
- Live Balance, Income & Expense Summary
- Transaction History with Search & Filters
- Delete Transactions
- Cash Flow Visualization (Chart.js)
- Multi-Currency Support (INR, USD, EUR, GBP, JPY)
- Profile Settings
- Dark Mode
- Local Storage Data Persistence
- Import & Export Transactions
- One-Click Reset Data
- Responsive Design

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6+)
- Chart.js
- Local Storage API

## 🚀 How to Run

1. Download or clone the project.
2. Open `login.html` in your web browser.
3. Register a new account.
4. Log in and start tracking your finances.

## 📌 Notes

- No backend or database is required.
- All user accounts and transactions are stored in the browser using Local Storage.
- Each user has separate financial data and preferences.
