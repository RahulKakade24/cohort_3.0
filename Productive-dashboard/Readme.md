# ⚡ Productive-DashBoard

ProductiveDashBoard is a premium, lightweight, single-page productivity dashboard designed to optimize your daily workflow. Built using native web technologies, it offers a sophisticated visual experience with smooth, dynamic transitions, context-aware components, and full support for both a sleek dark mode and a beautifully balanced, high-contrast light mode.

> [!TIP]
>
> ### 🌐 Live Demo
>
> Experience the active application here: [https://productivedashboard-chi.vercel.app/]

## ✨ Features

- **📊 Overview Dashboard:** Highlights a dynamic hero clock card, local weather feeds, and live completion statistics rendered via Chart.js.
- **📋 Task Management (To-Dos):** Create, filter (All, Active, Done), and prioritize high-urgency tasks with persistent local storage.
- **📅 Daily Execution Planner:** Hourly modular structure (6:00 AM – 11:00 PM) to layout and lock in your daily timeline.
- **⏳ Focus Timer:** A Pomodoro-style block timer featuring adjustable interval modes (Focus, Short Break, Long Break) and interactive session tracking dots.
- **🎯 Big Milestones (Goals):** Define long-term success conditions and track your achievements with a custom milestone interface.
- **🌓 Adaptive Dark/Light Mode:** Seamlessly swaps CSS theme tokens dynamically for maximum readability under any lighting condition.

---

## 🛠️ Tech Stack & Assets

ProductiveDashBoard uses modern front-end tooling loaded externally for instant deployment without complex compilation pipelines:

- **Core Architecture:** Semantic HTML5, CSS3 Custom Properties (Variables), and Modern Vanilla ES6+ JavaScript.
- **Styling Engine:** [Tailwind CSS v3](https://tailwindcss.com/) (Utility-first CSS framework framework layout scaffolding).
- **Typography:** [Plus Jakarta Sans](https://fonts.google.com/specimen/Plus+Jakarta+Sans) via Google Fonts.
- **Iconography:** [Font Awesome v6.5.1](https://fontawesome.com/) vector icon library.
- **Data Visualization:** [Chart.js](https://www.chartjs.org/) for rendering the responsive productivity bar chart metrics.
- **API Integration:** [OpenWeatherMap API](https://openweathermap.org/) (for localized environment reports).
